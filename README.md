# LabEx CLI

This is a command line interface for LabEx.

## Installation

Python 3.8 or higher is required.

Clone the repository:

```bash
git clone https://github.com/labex-labs/labex-cli.git
```

Install the package:

```bash
cd labex-cli
pip install --editable .
```

## Usage

```bash
labex --help
```

Create a new lab:

```bash
labex lab create
```
